mod_floatconvert.hsp
HSP 用拡張モジュール

対象 HSP バージョン: HSP 3.0 以降

作者 naznyark
Ver 2.00 ( 2008/06/12 )

//----------------------------------------------------------------------------

◎概要

HSP 用の拡張モジュールです。
float 型の値(単精度実数値、32bit)(のメモリイメージ)と double 型の値(倍精度実数値、64bit)の相互変換を可能にします。

　HSP プログラムから HSP 外部のコンポーネントを使用する際に float 型の値をメンバに持つ構造体などが必要になる場合があります。
HSP の基本機能には float 型引数を持つ外部命令・外部関数に double 型の値を変換して渡す機能はありますが float 型の値を直接操作する機能はありません。
このモジュールによる拡張機能では float 型の値と double 型の値の相互変換を行なうことができるようになり、間接的に float 型の値の操作を行なうことができるようになります。

◎アーカイブに含まれるファイル

○構成
　このアーカイブには以下の構成のファイル・フォルダが含まれています。

    mod_floatconvert_readme.txt
    common/
        mod_floatconvert.hsp
    docs/
        mod_floatconvert_readme.txt
    hsphelp/
        mod_floatconvert_help.hs
    sample/
        mod_floatconvert_sample/

○ファイル・フォルダの説明
・mod_floatconvert_readme.txt
    モジュールの基本的な説明書、今読んでいるこのファイルです。
    同名のファイルが二箇所にありますが同じ内容です。

・mod_floatconvert.hsp
    モジュールの本体です。
    HSP のインストールフォルダの中の common フォルダの中にコピーしてください。
    モジュールによる拡張機能を使用するにはソースコードの冒頭部分でこのファイルをインクルードする必要があります。

・mod_floatconvert_help.hs
    モジュールで使用可能になる命令等の説明が書かれたヘルプファイルです。
    HSP のインストールフォルダの中の hsphelp フォルダの中にコピーすると説明をヘルプブラウザで見ることができるようになります。

・mod_floatconvert_sample/
    モジュールを使ったサンプルプログラムの入ったフォルダです。

//----------------------------------------------------------------------------

◎インストール等

○インストール
　アーカイブを解凍してその中の全てのファイル・フォルダを HSP のインストールフォルダの中にコピーして下さい。
(モジュールによる拡張機能を使うだけならば mod_floatconvert.hsp を適切な場所にコピーすれば十分です。)

○アンインストール
　インストール時にコピーした全てのファイル、および、mod_floatconvert_sample フォルダを削除してください。

○モジュールによる拡張機能を使用可能にする
　拡張機能を使うプログラムのソースコード冒頭部分で

    #include "mod_floatconvert.hsp"

としてモジュールの本体ファイルをインクルードします。

○実行形式ファイル作成時の注意
　とくにありません。
